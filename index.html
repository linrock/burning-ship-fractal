Burning ship fractal

Images are generated by analyzing points on the complex plane:

Z_n+1 = (|Z_re| + i|Z_im|)^2 + (x0, y0)

We can explore fractals by putting different starting numbers into the equation and making note of:

  • whether the numbers diverge to infinity or stay within a limit
  • if they diverge
    • the number of iterations until reaching the escape radius
    • the magnitude of the escape

The number of iterations is determined by this equation:

const ESCAPE_THRESHOLD = 4;
const MAX_ITERATIONS = 255;

function iterateUntilEscape(x0, y0) {
  let x = 0;
  let y = 0;
  let iteration = 0;
  while ((x * x + y * y < ESCAPE_THRESHOLD) && (iteration < MAX_ITERATIONS)) {
    const x_n = x * x - y * y + x0;
    const y_n = 2 * Math.abs(x * y) + y0;
    x = x_n;
    y = y_n;
    iteration++;
  }
  return [iteration, xSq + ySq];
}
